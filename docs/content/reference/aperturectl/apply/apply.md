## aperturectl apply

Apply Aperture Policies

### Synopsis

Use this command to apply the Aperture Policies.

### Options

```
  -h, --help                 help for apply
      --kube-config string   Path to the Kubernetes cluster config. Defaults to '~/.kube/config' or $KUBECONFIG
```
